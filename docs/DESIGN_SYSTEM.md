# Wartell Studios — Design System

> Version 1.0 Foundation

## 1. Visual Direction

La identidad de Wartell Studios combina:

- Tecnología.
- Ciencia ficción.
- Diseño militar.
- Cyberpunk.
- Naturaleza.
- Arte urbano.
- Interfaces limpias.
- Experiencias inmersivas.

El resultado debe sentirse moderno, oscuro, premium y funcional.

### 8. Design Principles

La legibilidad está por encima de los efectos.
Las animaciones no deben bloquear la navegación.
Cada elemento debe tener una función.
La interfaz debe sentirse consistente.
El diseño móvil no debe ser una versión reducida del escritorio.
La identidad visual debe conservarse en todos los módulos.
